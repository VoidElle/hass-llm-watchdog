# LLM Watchdog docs

LLM Watchdog monitors the health of major AI/LLM providers from Home Assistant by combining public status pages with optional API probes.

## Pages

- [Installation](installation.md)
- [Configuration](configuration.md)
- [Sensors](sensors.md)
- [Automations](automations.md)
- [Troubleshooting](troubleshooting.md)

## Features

- 🏠 Native Home Assistant config flow and sensor entities
- 🌐 Passive monitoring from public provider status pages
- 🔑 Optional active API probes with your own API keys
- 📊 Per-provider sensors plus a worst-status summary sensor
- 🚨 Easy automation targets for alerts, dashboards, and digests
- ⚙️ Configurable polling interval with options-flow reconfiguration
